# Save to Desktop

An event photo sharing platform that makes it easy to collect, organize, and share photos from any event.

Guests can upload photos through shareable links, while event organizers manage submissions from a central dashboard.

## Features
- Shareable event upload links
- Guest photo uploads
- Photo approval workflow

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- AWS S3
- DynamoDB

Currently in development.
